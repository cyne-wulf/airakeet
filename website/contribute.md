# Airakeet: Download and Contribute

Download Airakeet, contribute on GitHub, or ask questions about the open-source local-first dictation app.

- Releases: https://github.com/cyne-wulf/airakeet/releases
- Repository: https://github.com/cyne-wulf/airakeet
- Contact: mailto:acynewulf@gmail.com

## Download and Source

GitHub Releases is the canonical place to grab the latest Airakeet build. The project is open source and built to be remixed, forked, and improved.

## How to Get Involved

Start by downloading a release, reading through the GitHub repository, and reaching out if you want to contribute ideas, fixes, docs, or questions.

- Download: get the latest build from GitHub Releases.
- Repository: explore the source, issues, and future improvements on GitHub.
- Privacy: Airakeet keeps transcription on-device and avoids cloud dependency.

## Good to Know

- Download: grab the latest app build from the GitHub Releases page.
- Contributing: explore the source, issues, and project history on GitHub.
- Questions: use email directly if you want to help, collaborate, or ask about the project.
- Privacy: Airakeet keeps audio and transcripts on-device, which makes it a strong fit for sensitive workflows.
- License: clone it, fork it, or ship it as-is. Airakeet is MIT licensed and built to be remixed.
