# Airakeet Landing Site

This directory hosts the static website for Airakeet, the open-source, local-first dictation app designed for 8GB Apple Silicon machines.

## Structure
- `index.html` – main landing page with product overview.
- `contribute.html` – download, contribution, and questions page.
- `showcase.html` – placeholder for upcoming workflow gallery.
- `styles.css` / `script.js` – shared styling and interactivity.
- `Final-icon.png`, `share-card.png` – iconography and Open Graph assets.

## Development
No build tooling is required. Update the HTML/CSS/JS files directly in `website/` and open a pull request.

## Deployment
1. Commit updates to `main` (or merge via PR).
2. The repo-level Pages workflow uploads `website/` and deploys it to https://cyne-wulf.github.io/airakeet/ and https://airakeet.com/.

## Canonical Repository
- `cyne-wulf/airakeet` - GitHub repository, releases, app source, and site content.
